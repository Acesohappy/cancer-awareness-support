# Cancer Awareness & Support

A simple React web page built to promote cancer awareness and provide emotional support.  
It includes an inspiring landing section, a contact form, and live motivational quotes fetched from a public API.

## Technologies Used
- React  
- HTML5 & CSS3  
- JavaScript (ES6)  
- [Real-Inspire API]([https://github.com/lukePeavey/quotable](https://github.com/mohamadayash22/real-inspire))
